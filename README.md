## Overview

The **Barangay E-Services Management System** is a computer-based application designed to streamline and modernize barangay operations by digitizing resident records and service requests. Instead of relying on manual logs and paper-based processes, this system securely stores resident information and service transactions in a structured database, improving efficiency, accuracy, and accessibility.

This project was developed as part of an academic requirement and demonstrates practical use of **Object-Oriented Programming (OOP)** concepts in Python.

---

## Features

- Manage resident profiles (CRUD operations)
- Track barangay service requests (e.g., certificates, clearances)
- Store and retrieve data using a relational database
- User-friendly command-line or GUI interface (depending on implementation)
